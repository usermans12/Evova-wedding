# Offline Serif Fonts (Kategori Klasik, Karakter & Kemewahan)

Folder ini mendokumentasikan pemetaan lokal dari font serif dekoratif mewah yang digunakan sebagai tajuk / nama mempelai pada tema undangan.

## Font yang Dipetakan:
- **Cinzel / Cinzel Decorative**: Dipetakan ke font lokal premium perangkat seperti `Georgia`, `Baskerville`, `Didot`, atau `Times New Roman`.
- **Cormorant Garamond**: Font novelis bergaya klasik Eropa dengan serif yang elegan dan tipis.
- **Playfair Display**: Font display serif kontras tinggi yang menghadirkan tampilan premium bagai majalah cetak mewah.

Dideklarasikan sepenuhnya tanpa `@import` online, sehingga muat secara instan (0ms delay) pada semua jenis operating system modern (iOS, Android, Windows, macOS).
