# Offline Script / Calligraphy Fonts (Kategori Kaligrafi & Tulisan Tangan)

Folder ini mendokumentasikan pemetaan lokal dari font bertipe script/cursive romantis yang digunakan untuk inisial nama mempelai atau pemanis estetika pada amplop digital atau pembuka undangan.

## Font yang Dipetakan:
- **Great Vibes**: Font tulisan sambung yang sangat leguler. Dipetakan ke core handwriting sistem seperti `SignPainter` (macOS), `Brush Script MT` / `Gabriola` (Windows), atau core cursive bawaan perangkat mobile.
- **Parisienne / Allura / Pinyon Script**: Font kaligrafi halus bergaya Perancis abad ke-18.

Dijalankan secara native pada runtime layout tanpa melakukan handshake TCP ke server Google Fonts.
