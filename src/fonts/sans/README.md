# Offline Sans-Serif Fonts (Kategori Modern & Geometris)

Folder ini mendokumentasikan pemetaan lokal dari font sans-serif yang digunakan di dalam seluruh tema undangan pernikahan digital kustom ini.

## Font yang Dipetakan:
- **Poppins**: Menggunakan font modern bawaan sistem perangkat seperti `SF Pro` (iOS/macOS), `Roboto` (Android), atau `Segoe UI` (Windows).
- **Inter**: Font standard UI modern, dipetakan ke font lokal instan.
- **Montserrat / Lato / Nunito / Open Sans**: Font penjelas isi / tulisan badan undangan.

Semua font dideklarasikan lewat pemetaan `@font-face` lokal di `/src/fonts/fonts.css`, sehingga tidak memerlukan jaringan internet eksternal saat memuat undangan pernikahan.
